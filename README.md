# Employee Management System

A desktop application built with Python to help company managers easily and securely manage employee records. 

---

## 🚀 Features
* **Complete CRUD Control:** Save, edit, and delete employee data effortlessly.
* **Security-Focused:** 
  * The application requires a **password** to open.
  * Editing or deleting an employee requires entering their unique **Secret Number**.
  * To protect privacy, this Secret Number is completely hidden from the main application screen—only managers who already know it can perform these actions.

## 🛠️ Tech Stack
* **Language:** Python
* **GUI Library:** Tkinter
* **Database:** SQLite3

## ⚙️ How to Run
1. Clone or download this repository.
2. Ensure you have Python installed.
